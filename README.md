# Islander Roofing and Siding

Live site for a Mid-Island roofing and siding contractor built with Vite, TailwindCSS, and vanilla JavaScript.

## Development

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run images:optimize`
- `npm run preview`
- `npm run test:unit`
- `npm run test:e2e`
- `npm run test`

## Output

- Source pages live in `src/`
- Production output is written to `docs/`
- Shared shell chrome is baked during the build step
- Keep copy, routes, and tests aligned when the live site changes
- Unit coverage runs with Vitest and jsdom
- Browser coverage runs with Playwright against the Vite dev server
